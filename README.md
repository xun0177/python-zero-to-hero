# Python Zero to Hero

从零开始，用练习题掌握 Python 基础语法。

本系列教程面向零基础初学者，每道题聚焦一个核心知识点，由浅入深，循序渐进。每道题都配有详细的代码注释和技术说明，帮助你不仅学会"怎么写"，更理解"为什么这么写"。


## 📖 适合人群

- 完全零基础的编程新手
- 有其他语言基础、想快速入门 Python 的学习者
- 需要系统复习 Python 基础知识的开发者


## 🚀 使用方法

```bash
# 克隆项目到本地
git clone https://github.com/xun0177/python-zero-to-hero.git

# 进入项目目录
cd python-zero-to-hero

# 直接运行任意题目（例如第 1 题）
python 01_function_intro.py
每道题都是独立的 `.py` 文件，可以直接运行查看效果。建议按顺序开始学习。
```

## ✨ 代码特色

每道题都包含两部分：

1. **代码实现**：带详细中文注释，逐行解释关键语法。
2. **补充说明**：系统总结知识点、常见注意事项和扩展思考题。

你可以先运行代码观察输出，再结合注释和补充说明深入理解。
